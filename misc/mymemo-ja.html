Cygwin+MinGWでのビルド環境の設定方法

  • Cygwinのセットアップ時に、MinGWのgccとMinGWのzlibも選択する。
  • MinGWのregexは http://gnuwin32.sourceforge.net/packages/regex-spencer.htm から手に入れる。そして、*.h は /usr/include/mingw の下に、*.lib と *.a と *.dll.a は /lib/mingw の下に、*.dll は /binの下に移動させる。
  • MinGWのlibiconvは http://www.mingw.org/ から手に入れる。そして、*.h は /usr/include/mingw の下に、*.dll.a は /lib/mingw の下に、*.dll は /binの下に移動させる。
  • Pthreads for Win32は http://sources.redhat.com/pthreads-win32/ から手に入れる。そして、*.h は /usr/include/mingw の下に、*.a は /lib/mingw の下に、*.dll は /binの下に移動させる。
  • xdoc2txtは http://www31.ocn.ne.jp/~h_ishida/xdoc2txt.html から手に入れる。/usr/local/xdoc2txt を作成してアーカイブの一式をその中に移動させる。
  • QDBMのインストールは ./configure --enable-zlib --enable-iconv ; make mingw ; make install-win で行う。
  • Hyper Estraierのインストールは ./configure ; make win ; make install-win で行う。パッケージ作成は make winpkg で行う。
